Codeology

Code is beautiful.

The complexity and brilliance of developers' work sometimes goes unrecognized. Codeology reimagines their code as the art it really is.

This open-source web app, made possible by Braintree, transforms public Github projects into 3D ASCII creatures. And determines the size, shapes, and colors based on the coding languages used in each project.
